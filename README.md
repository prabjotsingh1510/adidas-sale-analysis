# Adidas Sale Analysis
The dashboard provides a comprehensive overview of key business metrics such as Total Sales ($900M), Units Sold (2M), Operating Profit ($332M), Operating Margin (42%), and Average Price per Unit ($45). Interactive visuals enable users to explore monthly sales trends, state-wise and region-wise performance, top-selling products, and leading retailers.

Key Features:

Interactive filters for region and invoice date

Monthly sales trend analysis

Geographic sales visualization by state and region

Product category performance comparison

Retailer-wise sales contribution analysis

Tools & Technologies:
Power BI, SQL/PostgreSQL, Excel, Data Cleaning, DAX

Outcome:
The project helps identify high-performing regions, top product categories, and major retail partners, supporting data-driven decision-making for marketing, inventory planning, and sales strategy.
